ⓘ Integration Guide

What this builder does: Creates ui.json — the launcher configuration and declarative screen definitions.
What it doesn't do: Create Python apps. For custom logic (network calls, dynamic lists, keyboards), build a Python app instead. See custom app docs.
After export:
  1. Save ui.json to device flash root
  2. Deploy any catalog app .py files
  3. Reset the board
Full details: manual UI pages guide.
Device paths: All apps live in apps/ on the flash. Built-in entries use format module:function (e.g., wifi_demo:HomeScreen). Module my_app is imported as apps.my_app.
Hooks: Button actions like hook:say_hi call Python callbacks registered in src/user_hooks.py. Give widgets an id so hooks can find and update them.
